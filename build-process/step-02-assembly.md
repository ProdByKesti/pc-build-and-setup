# Step 2 - PC Assembly

## Components Installed
- CPU: AMD Ryzen 5 5600X
- Motherboard: ASUS B550
- RAM: 32 GB DDR4
- PSU: 750W Modular
- GPU: NVIDIA RTX 3070

## Assembly Process
1. Installed CPU into the motherboard socket
2. Applied thermal paste and mounted the CPU cooler
3. Installed RAM in dual-channel configuration
4. Installed GPU into the Motherboard socket
5. Mounted Motherboard into the case
6. Connected power cables and front -panel connectors

## Issue Encountering
- Tight cable routing in the case
- GPU clearance required adjustment

## Lessons Learned
- Modular PSUs make cable management easier
- Always check GPU length vs case size
