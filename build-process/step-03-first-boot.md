# Step 03 - First Boot and Initial Checks

## First Power-On
The system powered on successfully on the first attempt .
All fans started spinning and no error LEDs were visible on the motherboard.

## BIOS Access
Entered BIOS using the DEL key.

Verified:
- CPU detection correctly
- RAM detected at default speed
- ALL storage devices visible
- CPU temperature in normal range

## Initial Issues 
- RAM was running at base frequency
- Boot order needed adjustment

## Actions Taken
- Enabled DOCP profile for RAM
- Adjusted boot priority
- Saved BIOS settings and rebooted

## Result

System rebooted successfully with updated settings.
No stability issues observed at this stage.
